# Adding TEKNO and OMNISPHERE to Your Template

## Quick Steps

### 1. Open Your Template
```bash
open -a "Ableton Live 12 Suite" "Techno-Template-Output/Manual-set Project/Manual-set.als"
```

### 2. Add TEKNO to Bass Tracks (Tracks 6 & 7)

**Track 6 - BASS TEKNO:**
1. Click on the track "06 - BASS TEKNO"
2. In the browser, navigate to: **Plug-ins ‚Üí VST3 ‚Üí TEKNO**
3. Drag TEKNO onto the track's device chain
4. Select a bass preset:
   - "Deep Sub Bass"
   - "Minimal Techno Bass"
   - "Dark Rolling Bass"
5. Adjust the cutoff and resonance for movement

**Track 7 - ACID BASS TEKNO:**
1. Click on "07 - ACID BASS TEKNO"
2. Drag TEKNO VST3 onto this track
3. Select an acid preset:
   - "303 Classic"
   - "Squelchy Acid"
   - "Resonant Bass"
4. Enable filter modulation for that classic acid sound

### 3. Add OMNISPHERE to Synth Tracks (Tracks 8 & 9)

**Track 8 - LEAD OMNISPHERE:**
1. Click on "08 - LEAD OMNISPHERE"
2. In browser: **Plug-ins ‚Üí VST3 ‚Üí Omnisphere**
3. Drag Omnisphere onto the track
4. Load a lead preset:
   - Browse: **Synthesizer ‚Üí Lead**
   - Try: "Tech Lead Mono", "Minimal Lead", "Dark Lead"
5. Adjust envelope and filter to taste

**Track 9 - STABS OMNISPHERE:**
1. Click on "09 - STABS OMNISPHERE"
2. Drag Omnisphere VST3 onto the track
3. Load a stab/chord preset:
   - Browse: **Synthesizer ‚Üí Pads & Strings** or **Arpeggiator**
   - Try: "Techno Stab", "Analog Chord", "Saw Stab"
4. Set short decay for punchy stabs

### 4. Add Drums to Percussion Tracks (Tracks 1-5)

For the drum tracks, you have options:

**Option A: Use Drum Rack (Recommended)**
1. Select tracks 1-5
2. Drag **Drum Rack** from the browser
3. Drag samples into the Drum Rack pads:
   - Track 1 (Kick): C1 pad
   - Track 3 (Clap): D1 pad
   - Track 4 (HH Closed): F#1 pad
   - Track 5 (HH Open): A#1 pad

**Option B: Use Individual Samplers**
1. Drag **Simpler** onto each percussion track
2. Load drum samples from your library
3. Adjust sample start, envelope, and filter

**Option C: Use Battery or Kontakt (if you have them)**
1. Drag Battery/Kontakt VST3 onto a single track
2. Load a techno drum kit
3. MIDI note assignment already matches standard GM mapping

### 5. Save as Template

Once you've added all instruments:

1. **File ‚Üí Save Live Set As...**
2. Save as: `Techno-Banger-Complete.als`
3. **File ‚Üí Save Live Set as Template**
4. Template name: `Techno Banger - Full VST3`
5. Template folder: Choose category (e.g., "Electronic")

### 6. Test Your Template

1. **File ‚Üí Close Live Set**
2. **File ‚Üí New Live Set from Template**
3. Select "Techno Banger - Full VST3"
4. Click all MIDI clips to hear your patterns with instruments!

## Recommended Settings

### TEKNO Settings
- **Bass Tracks (6 & 7):**
  - Filter: Low-pass 24dB
  - Cutoff: 200-800 Hz (automate this!)
  - Resonance: 20-40%
  - Envelope: Short attack, medium decay
  - Sub oscillator: ON (for extra low end)

### OMNISPHERE Settings
- **Lead Track (8):**
  - Mono mode for fat leads
  - Portamento/Glide: 20-50ms
  - Filter envelope modulation
  - Add chorus or unison for width

- **Stab Track (9):**
  - Envelope: Fast attack, short decay
  - Layer: 2-3 oscillators for thickness
  - Detune slightly for width
  - Consider adding reverb send

## Mixing Chain Suggestion

After adding instruments, add these effects:

### On Each Track:
1. **EQ Eight** - Cut unnecessary frequencies
2. **Compressor** - Glue the sound
3. **Saturator** (optional) - Add harmonics

### Master Chain:
1. **Glue Compressor** - Light compression (2:1 ratio)
2. **EQ Eight** - Final tone shaping
3. **Limiter** - Prevent clipping (-0.3dB ceiling)
