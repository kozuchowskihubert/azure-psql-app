## Getting Started

### Basic Workflow

```
1. Select Root Note (A, C, E, etc.)
2. Choose Scale Mode (Natural Minor, Phrygian, etc.)
3. Select Techno Subgenre (Acid, Minimal, Industrial, etc.)
4. Click "‚ö° Generate Sequence"
5. Choose Synthesis Engine (Acid Bass, Minimal Stab, etc.)
6. Adjust Modulation Parameters (Resonance, Cutoff, Glide)
7. Click "‚ñ∂Ô∏è Play Preview"
8. Create drum pattern in 16-step sequencer
9. Play complete techno track
```

### Quick Start Tips

- **Acid Techno**: Use Acid Bass engine with high resonance (70-90%)
- **Minimal Techno**: Minimal Stab with low cutoff (100-300 Hz)
- **Hard Techno**: Industrial Drone with high distortion (50-80%)
- **Dub Techno**: Dub Chord with heavy reverb and delay
- **Detroit Techno**: Detroit Pad with medium cutoff (500-1000 Hz)

## Techno Subgenres

### 1. üî• Hard Techno

**Characteristics:**
- **BPM**: 140-150
- **Energy**: High, aggressive, relentless
- **Kick**: Distorted, punchy, 4-on-the-floor
- **Bassline**: Fast, driving, repetitive
- **Hi-hats**: Constant 16th notes

**Sequence Pattern:**
```
[Root, Root, Fifth, Root, Third, Root, Seventh, Root]
Rhythm: Driving, relentless
```

**Production Tips:**
- Use high distortion (50-80%)
- Minimal breaks, maximum energy
- Layer multiple percussion sounds
- Hard clipping and compression

**Famous Artists:** Charlotte de Witte, I Hate Models, Amelie Lens

---

### 2. üîä Acid Techno

**Characteristics:**
- **BPM**: 130-140
- **Sound**: TB-303 acid basslines
- **Filter**: Sweeping resonant filter
- **Pattern**: Hypnotic, repetitive sequences
- **Vibe**: Psychedelic, groovy

**Sequence Pattern:**
```
[Root, Third, Fifth, Third, Seventh, Fifth, Third, Root]
Rhythm: Hypnotic, squelchy
```

**Synthesis:**
- **Resonance**: 70-95% (classic acid sound)
- **Filter Sweep**: Automated envelope modulation
- **Glide**: 30-100ms (portamento slides)
- **Waveform**: Sawtooth or square

**Production Tips:**
- Automate filter cutoff and resonance
- Use glide/portamento between notes
- Add distortion for extra grit
- Layer with percussion for groove

**Famous Artists:** Plastikman (Richie Hawtin), DJ Pierre, Phuture

### 3. üéµ Minimal Techno

Rhythm: Sparse, spacious

- Focus on micro-timing and groove

## Scale Modes

### Natural Minor (Aeolian)
**Intervals:** 0, 2, 3, 5, 7, 8, 10  
**Character:** Dark, melancholic, sad  
**Use:** Most versatile, works for all subgenres  
**Example:** A minor = A, B, C, D, E, F, G

### Phrygian
**Intervals:** 0, 1, 3, 5, 7, 8, 10  
**Character:** Spanish, exotic, dark  
**Use:** Industrial, acid techno  
**Example:** E Phrygian = E, F, G, A, B, C, D

### Dorian
**Intervals:** 0, 2, 3, 5, 7, 9, 10  
**Character:** Groovy, jazzy, funky  
**Use:** Detroit techno, minimal  
**Example:** D Dorian = D, E, F, G, A, B, C

### Minor Pentatonic
**Intervals:** 0, 3, 5, 7, 10  
**Character:** Simple, hypnotic  
**Use:** Minimal techno, acid  
**Example:** A minor pentatonic = A, C, D, E, G

### Harmonic Minor
**Intervals:** 0, 2, 3, 5, 7, 8, 11  
**Character:** Exotic, dramatic, Middle Eastern  
**Use:** Industrial, experimental  
**Example:** A harmonic minor = A, B, C, D, E, F, G#

## Modulation & Effects

### Filter Controls

#### Resonance (0-100%)
- **Low (0-30%)**: Gentle filtering
- **Medium (30-70%)**: Classic techno sound
- **High (70-100%)**: Acid squelch, self-oscillation

**Sweet Spots:**
- Acid: 70-90%
- Minimal: 20-40%
- Industrial: 50-70%

#### Cutoff (100-5000 Hz)
- **Low (100-500 Hz)**: Deep, bassy
- **Medium (500-2000 Hz)**: Balanced
- **High (2000-5000 Hz)**: Bright, aggressive

**Sweet Spots:**
- Dub: 300-800 Hz
- Acid: Swept 200-2000 Hz
- Hard: 1000-3000 Hz

### Performance Controls

#### Glide/Portamento (0-200ms)
- **0ms**: No glide, precise
- **30-50ms**: Subtle slides
- **100-200ms**: Dramatic swoops

**Usage:**
- Acid basslines: 30-100ms
- Minimal: 0-20ms
- Industrial: 0ms (precise)

#### Distortion (0-100%)
- **Low (0-30%)**: Subtle warmth
- **Medium (30-60%)**: Character and grit
- **High (60-100%)**: Extreme saturation

**Usage:**
- Clean: 0-20%
- Acid: 30-50%
- Industrial: 60-100%

### Effects

#### ‚è±Ô∏è Delay
- **Type**: Stereo ping-pong
- **Time**: Dotted eighth (375ms at 120 BPM)
- **Feedback**: 30%
- **Mix**: 40% wet

**Best For:** Dub techno, spacious mixes

#### üåä Reverb
- **Type**: Algorithmic plate
- **Time**: 2 seconds
- **Mix**: 30% wet
- **Character**: Smooth, evolving

**Best For:** Pads, atmospheric elements

#### „Ä∞Ô∏è LFO Modulation
- **Rate**: Sync to tempo (1/4, 1/8, 1/16)
- **Depth**: Adjustable
- **Target**: Filter cutoff, resonance, amplitude

**Best For:** Movement, evolution
