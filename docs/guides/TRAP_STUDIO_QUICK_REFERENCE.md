| Parameter | Range | Default | Function | Pro Tip |

| **Decay** | 0.1-2.0s | 0.5s | Tail length | Short = kicks, Long = rolling bass |

| **Hyperpop** | 150-180 | N/A | 165 BPM |
| **Jersey Club** | 130-145 | 65-72 BPM feel | 140 BPM |

### Double-Time Explained
Many trap beats use **half-time drums** at high BPM:
- **140 BPM trap** = feels like 70 BPM (snare on 3)
- **150 BPM drill** = feels like 75 BPM (slower groove)
- This creates the signature "bouncy" trap feel

## ğŸ“ Pattern Building Tips

### Kick Drum Philosophy
- **Classic trap**: Sparse kicks (let 808 dominate)
- **Drill**: Minimal kicks [0,6,10] signature pattern
- **Boom-bap**: Four-on-floor [0,4,8,12]
- **Hyperpop**: Complex/chaotic (double/triplet)

### Hi-Hat Strategy
- **Start simple**: Closed 8ths pattern
- **Add energy**: Trap rolls in last bar
- **Drill signature**: Full 16th note rolls
- **Boom-bap**: Shuffled or half-time swing

### 808 Bass Integration
- **Complement kicks**: Don't double every kick
- **Melody**: Use pitch glide for slides
- **Rhythm**: Add syncopation (off-beat 808s)
- **Sub focus**: Keep frequency 40-60 Hz

### Chord Progression Usage
- **Dark trap**: Minor progressions (i-VI-III-VII)
- **Emotional**: Use v and VI for sadness
- **Drill**: Stay dark (i-III-VII-VI)
- **Cloud rap**: Major chords for dreaminess
