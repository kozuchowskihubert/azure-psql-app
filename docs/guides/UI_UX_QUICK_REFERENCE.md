## ðŸ”§ Troubleshooting

### Common Questions

**Q: Tooltips not showing?**
- Make sure you hover over the â„¹ icon
- On mobile, tap instead of hover
- Check browser supports CSS tooltips

**Q: Guidance panel disappeared?**
- Check bottom-right corner
- It might be minimized (+ icon)
- Click header to expand

**Q: Style indicators not updating?**
- Ensure JavaScript is enabled
- Try refreshing the page
- Check browser console for errors

**Q: Quick actions not working?**
- Make sure audio context is initialized
- Click "Play" first to enable audio
- Check preset functions are defined

**Q: Values not matching style label?**
- Labels update on slider release
- Try clicking elsewhere after adjusting
- Refresh if persistent
