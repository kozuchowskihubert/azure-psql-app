## Technical Details

### Synthesis Techniques Used

1. **Subtractive Synthesis** - Traditional VCO → VCF → VCA signal flow
2. **FM Synthesis** - Oscillator frequency modulation
3. **Hard Sync** - Oscillator synchronization
4. **Ring Modulation** - Amplitude modulation between oscillators
5. **PWM** - Pulse width modulation
6. **S&H Sequencing** - Sample and hold for random sequences
7. **LFO Modulation** - Low frequency oscillator routing
8. **Envelope Modulation** - ADSR envelope routing
9. **Noise Synthesis** - White/pink noise for percussion and textures
10. **Cross-Modulation** - Complex modulation routings

### Patch Cable Color Coding

- **Red** - Audio signal path
- **Blue** - Amplitude modulation (VCA)
- **Green** - Filter modulation
- **Yellow** - Pitch/frequency modulation
- **Purple** - Special routing

### Module Parameters

#### VCO (Voltage Controlled Oscillator)
- Frequency: 20 Hz - 20 kHz
- Waveforms: Sine, Triangle, Sawtooth, Pulse
- Modulation inputs: FM, Sync, PWM

#### VCF (Voltage Controlled Filter)
- Cutoff: 0.0 - 1.0 (normalized)
- Resonance: 0.0 - 1.0
- Modes: LP (lowpass), HP (highpass), BP (bandpass)

#### VCA (Voltage Controlled Amplifier)
- Gain control via envelope or manual
- CV input for modulation

#### Envelopes
- Attack: 0.001s - 5.0s
- Decay: 0.001s - 5.0s
- Sustain: 0.0 - 1.0
- Release: 0.001s - 5.0s

#### LFO (Low Frequency Oscillator)
- Rate: 0.01 Hz - 20 Hz
- Waveforms: Sine, Triangle, Sawtooth, Square, Random
- Depth: 0.0 - 1.0
