# MIDI Demo Scripts

This directory contains scripts for running MIDI demonstrations and Ableton Live automation.

## Scripts

- **demo-interactive.sh** - Interactive MIDI generation demo
- **launch_deep_techno.sh** - Launch deep techno MIDI project
- **cleanup.sh** - Cleanup generated files

## Usage

### Run Interactive Demo
```bash
./demo-interactive.sh
```

### Launch Ableton Project
```bash
./launch_deep_techno.sh
```
