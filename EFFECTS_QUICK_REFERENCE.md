## ğŸ’¡ Pro Tips

### Mixing Tips
1. **Start subtle** - Add effects gradually (less is more!)
2. **Reverb first** - Adds space without changing tone
3. **Filter shapes** - Use before distortion for control
4. **Compressor glues** - Already on and optimized!
5. **Test in context** - Solo vs. full mix sounds different

### Creative Tips
1. **Automate LFOs** - Turn on/off during playback for build-ups
2. **Combine effects** - Reverb + Delay = huge space
3. **Extreme settings** - Try 100% values for sound design
4. **Layer LFOs** - Multiple LFOs = complex movement
5. **Effect order matters** - Distortion before reverb vs. after = different sounds

### Performance Tips
1. **CPU conscious** - Disable unused effects
2. **Batch changes** - Adjust multiple parameters before playing
3. **Monitor levels** - Effects can increase volume
4. **Save settings** - Note down combinations you like

## ğŸ†˜ Troubleshooting

### Effect not working?
1. Check toggle button says **[ON]** (green)
2. Verify parameter isn't at 0%
3. Try toggling off and on again
4. Refresh page if issues persist

### LFO not modulating?
1. Check toggle says **[ON]** (orange/green)
2. Increase depth parameter
3. Verify rate isn't too slow (try 1 Hz)
4. Check browser console for errors

### Audio clicking/popping?
1. Disable some effects to reduce CPU
2. Close other browser tabs
3. Lower distortion drive amount
4. Check system audio settings

### No sound at all?
1. Check browser permissions (allow audio)
2. Verify system volume is up
3. Try clicking page first (browser security)
4. Refresh page to reset audio context
